What it is

About the project

A rocker-bogie suspension rover chassis adapted from the Sawppy Rover project. The original uses LX-16A smart servos for both steering and drive. This build replaces them with DC gear motors and standard PWM servos that are locally available in Bangladesh at a fraction of the cost.

The chassis is designed to be the base for multiple rover applications. Swap the payload on top, keep everything underneath the same.

Reference Sawppy Rover by Roger Random, base chassis architecture this project is built on
